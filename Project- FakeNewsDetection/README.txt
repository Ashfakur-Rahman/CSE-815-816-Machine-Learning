To run the project:
1. Download the full project 'FakeNewsDetection' in a single folder as it is.
2. Download the data sets "Fake.csv" and "True.csv" in the same folder.
3. To download the dataset Follow this kaggle link: https://www.kaggle.com/datasets/emineyetm/fake-news-detection-datasets
4. Open this folder in Jupyter Notebook
5. Run the 'app.ipynb' file , click the link generated in the output.
6. input a demo news from 'news_for_test.docx' file.
7. Sumbit and see the result is Fake or True!!
